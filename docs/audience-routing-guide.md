# Audience routing guide (what to read / use next)

This repo contains:
- a **factual claim set** (see [`claims.md`](../claims.md), with claim IDs),
- a **debate + verdict record** (Issue #12 and record packets),
- and a **governance / litigation / legislation package** designed to be reusable.

This guide routes different audiences to the smallest set of documents that will get them to “usable” quickly.

> If you only have a few minutes: start with the time‑budget paths in [`docs/ai-digest-intro-blurb-gpt-5-1.md`](ai-digest-intro-blurb-gpt-5-1.md).
